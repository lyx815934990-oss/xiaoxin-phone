# 📱 小馨手机插件

一个为 SillyTavern 设计的沉浸式手机界面插件，让你的角色卡拥有真实的手机交互体验！

## ✨ 功能特色

- 📲 **微信聊天** - 完整的微信聊天界面，支持文字、图片、表情包、红包、转账等功能
- 📞 **电话通话** - 模拟真实的电话接听体验
- 💬 **短信功能** - 传统短信收发
- 🐦 **微博动态** - 发布和浏览微博动态
- 🎨 **自定义背景** - 支持自定义手机壁纸和应用背景
- 🖼️ **生图API** - 集成生图API，支持AI生成图片
- 💾 **数据独立** - 每张角色卡的数据独立存储，互不干扰
- 🔔 **实时监听** - 实时监听AI回复，自动解析并显示在手机界面中

## 📦 安装方法

### 通过 GitHub 导入

1. 打开 SillyTavern，进入 **扩展设置** → **第三方扩展**
2. 点击 **从 GitHub 导入**
3. 输入以下 GitHub 仓库地址：
   ```
   https://github.com/lyx815934990-oss/xiaoxin-phone.git
   ```
4. 点击 **导入**，等待安装完成
5. 刷新页面，插件即可使用

## 🚀 使用说明

1. 安装完成后，在聊天界面会看到一个悬浮的手机图标
2. 点击手机图标打开手机界面
3. 开始与角色进行手机交互！
